========================================
   OptiBoost - 사용법  (신급 에디션)
========================================

■ 설치 방법 (컴퓨터에 정식 설치)
  - "설치.bat" 더블클릭  →  시작 메뉴 + 바탕화면에 "OptiBoost" 바로가기 생성.
  - 파이썬 없이도 돌아가는 단독 실행파일(exe)로 설치됩니다.
  - 제거: [설정 > 앱 > 설치된 앱]에서 "OptiBoost" 제거,
          또는 "제거.bat" 실행.

■ 그냥 실행만 하려면
  - "실행.bat" 더블클릭 (설치 없이 바로 실행, 파이썬 필요).
  - 창이 안 뜨면 "오류확인용.bat" 으로 오류 확인.
  - ★ 최대 성능을 원하면 우측 상단 "관리자로 실행" 을 누르세요.
    (대기메모리 퍼지, 시스템 청소, 트윅 일부, 복구 도구가 활성화됩니다)

■ 필요 사항
  - Windows 10/11 + Python 3 (이미 설치됨). 추가 설치 필요 없음.


■ 탭별 기능 -------------------------------------------

[📊 대시보드]
  - CPU / 메모리 / 디스크(C:) 실시간 원형 게이지.
  - 🩺 PC 건강 점수 : 메모리·디스크 여유·시작앱 수·가동시간·임시파일을
    종합해 S~D 등급으로 진단. 문제점을 한 줄로 알려줍니다.
  - ⚡ 슈퍼 최적화 : 임시·캐시 정리 + 휴지통 + 메모리 정리
    + 대기메모리 퍼지(관리자) + DNS 정리를 한 번에!
  - ⏱ 자동 반복 최적화 : "시간/분/초" 간격마다 자동 최적화.
    (포함 항목 선택 가능, 카운트다운 표시, 설정 저장·자동 재개)
  - 🖥 백그라운드 실행 :
      · "Windows 시작 시 자동 실행" 체크 → 부팅하면 트레이에 자동으로 뜸.
      · "닫기(X) → 트레이로 숨기기" 체크(기본) → X 눌러도 종료 안 되고
        시계 옆 트레이로 숨어 백그라운드 계속 실행 (자동 반복 최적화 유지).
      · 트레이 아이콘 좌클릭 = 다시 열기 / 우클릭 = 메뉴(열기·최적화·종료).
      · 완전히 끄려면 트레이 우클릭 → "종료".

[🧹 청소]  ─ 딥클린 19종
  기본: 임시파일, 인터넷 캐시, 오류덤프, 블루스크린 덤프, 썸네일 캐시,
        윈도우 로그, 프리페치, 최근문서 기록
  GPU:  DirectX / NVIDIA / AMD 셰이더 캐시
  앱:   Chrome·Edge(모든 프로필), Discord, Spotify 캐시
  기타: 윈도우 업데이트 캐시, 전달 최적화, 휴지통
  ※ 내 PC에 존재하는 항목만 자동으로 표시됩니다. 스크롤해서 보세요.
  ▶ 매주 자동청소 예약도 여기서 켭니다.

[🚀 부스터]
  - 메모리 정리 : 워킹셋 정리 + (관리자) 대기메모리 퍼지 + 파일캐시 플러시
    → RAMMap 급의 강력한 메모리 회수!
  - 전원 옵션 고성능/균형, DNS 캐시 비우기, 백그라운드 프로세스 종료.

[🎮 게임]
  게임 부스트: 고성능 전원 + 게임모드 + 메모리(대기메모리 포함) 정리
  + 선택한 백그라운드 앱 닫기. 종료하면 전원 설정 자동 복원.

[🛡 보안]  ★ 바이러스 검사
  Windows 내장 백신(Microsoft Defender)으로 실제 악성코드를 검사합니다.
  - 상단: 실시간 보호 켜짐/꺼짐, 백신 정의 날짜, 마지막 검사 시각.
  - ⚡ 빠른 검사 : 자주 감염되는 위치만 검사 (보통 1~3분).
  - 🔍 전체 검사 : 디스크 전체 검사 (수십 분).
  - 🔃 백신 정의 업데이트 : 최신 바이러스 정보 내려받기.
  - 📋 위협 기록 : 지금까지 탐지·격리된 위협 목록.
  - ⏹ 중지 : 진행 중인 검사 멈추기.
  ※ 검사 결과는 아래 로그창에 실시간 표시. 위협 발견 시 경고가 뜹니다.
  ※ 다른 백신(V3, 알약 등)을 쓰면 그 프로그램이 주력일 수 있어요.

[🔥 트윅]  ★ 신급 기능
  성능 트윅 9종. 전부 백업 후 적용되어 "되돌리기" 한 번이면 원상복구!
  - 🏆 궁극의 성능 전원 모드 (숨겨진 Ultimate Performance 해금)
  - 🎥 게임 DVR·백그라운드 녹화 끄기 (FPS ↑)
  - 🌐 네트워크 지연 감소 - Nagle 끄기 (게임 핑 ↓) 🛡
  - 🖱 마우스 가속 끄기 (에임 일관성)
  - ✨ 시각효과 최고 성능 / 🪟 투명 효과 끄기
  - 🕵 진단 데이터·광고 ID 차단
  - ⚙ SysMain(Superfetch) 끄기 🛡 / 💤 최대 절전 끄기(수 GB 확보) 🛡
  "🔥 추천 모두 적용" 으로 한 방에 적용 가능. (🛡 = 관리자 필요)

[🩺 복구]  ★ 신급 기능  (관리자 필요)
  윈도우가 이상할 때 쓰는 공식 복구 도구 모음. 실시간 로그 표시.
  - 🔎 SFC : 손상된 시스템 파일 검사·복구
  - 🛠 DISM : 윈도우 이미지 복구 (SFC가 실패할 때)
  - 💽 디스크 검사 (chkdsk)
  - 🌐 네트워크 초기화 (인터넷 고장났을 때, 재부팅 필요)

[🧩 시작앱]  부팅 자동실행 켜기/끄기 (되돌리기 가능)
[💽 디스크]  큰 파일/큰 폴더 상위 300개 → 휴지통으로 삭제
[🔁 중복]    내용이 같은 중복 파일 찾기 → 휴지통으로 삭제


■ 안전 -------------------------------------------------
  - 트윅은 전부 백업 후 적용 → "되돌리기"로 100% 원상복구.
  - 파일 삭제(디스크/중복)는 휴지통행 → 복구 가능.
  - 시작앱 끄기도 백업됨 → 다시 켜기 가능.
  - 청소는 임시/캐시만 대상, 사용 중 파일은 자동 건너뜀.
  - 셰이더 캐시 삭제 후 첫 게임 실행은 로딩이 약간 느릴 수 있어요(재생성).
