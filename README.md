# OptiBoost (PC Optimizer)

윈도우용 올인원 최적화 프로그램. 파이썬 없이 **exe 하나로 실행**됩니다.
추가 설치·의존성 없이 파이썬 표준 라이브러리(tkinter, ctypes, winreg)만으로 만들었습니다.

> Windows 10/11 (64비트) 전용

---

## ⬇️ 다운로드 (그냥 쓰실 분)

1. 오른쪽 **[Releases](../../releases)** 에서 최신 `OptiBoost.exe` 를 받으세요.
2. 더블클릭하면 바로 실행됩니다. (설치·파이썬 불필요)

> **"Windows의 PC 보호" 파란 경고가 뜨면?**
> 인터넷에서 받은 미서명 프로그램이라 뜨는 정상 경고입니다.
> **[추가 정보] → [실행]** 을 누르면 됩니다. (백신이 막으면 '허용')

정식 설치(시작 메뉴·바탕화면 바로가기)를 원하면 릴리스의 `OptiBoost.zip` 을 받아
압축을 풀고 `설치.bat` 을 실행하세요. 제거는 [설정 > 앱 > 설치된 앱] 또는 `제거.bat`.

---

## ✨ 기능 (10개 탭)

- **📊 대시보드** — CPU/RAM/디스크 실시간 게이지, PC 건강 점수, 원클릭 슈퍼 최적화, 자동 반복 최적화, 백그라운드(트레이) 실행
- **🧹 청소** — 임시파일·캐시·셰이더캐시·휴지통 등 딥클린 19종 + 매주 자동청소 예약
- **🚀 부스터** — 메모리 정리(대기메모리 퍼지 포함), 전원 옵션, DNS 정리, 프로세스 종료
- **🎮 게임** — 게임 부스트(고성능+게임모드+앱정리, 종료 시 자동 복원)
- **🛡 보안** — Windows Defender로 바이러스 빠른/전체 검사, 정의 업데이트, 위협 기록
- **🔥 트윅** — 성능 트윅 9종(전부 백업·되돌리기 가능)
- **🩺 복구** — SFC / DISM / CHKDSK / 네트워크 초기화 (실시간 로그)
- **🧩 시작앱** — 부팅 자동실행 켜기/끄기
- **💽 디스크** — 큰 파일·큰 폴더 찾기
- **🔁 중복** — 중복 파일 찾기

대부분의 삭제·변경은 **휴지통 이동 / 백업 후 적용**이라 되돌릴 수 있습니다.
관리자 권한으로 실행하면 대기메모리 정리·시스템 청소·트윅·복구 기능이 전부 활성화됩니다.

## 🛠 소스로 직접 실행 / 빌드

```bash
# 실행 (파이썬 3 필요)
python PCOptimizer.pyw

# exe 빌드 (PyInstaller)
pip install pyinstaller
python -m PyInstaller --onefile --noconsole --icon icon.ico ^
    --add-data "icon.ico;." --name PCOptimizer PCOptimizer.pyw
```
